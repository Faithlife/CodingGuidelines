# dotnet-common-contributing

Publishes shared package repository contributor guidance to `CONTRIBUTING.md`.

## Behavior

This convention publishes shared release and prerelease guidance for Faithlife .NET library repositories. The file is copied as a whole because it is repository-level contributor documentation rather than local configuration.

## Example

```yaml
conventions:
  - path: Faithlife/CodingGuidelines/conventions/dotnet-common-contributing
```
