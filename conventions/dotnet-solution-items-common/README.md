# dotnet-solution-items-common

Adds common repository files and GitHub configuration files as .NET solution items.

## Behavior

The convention applies [dotnet-solution-items](../dotnet-solution-items/README.md) with the standard set of repository-root and `.github` paths that Faithlife .NET repositories should keep visible in the solution.

## Example

```yaml
conventions:
  - path: Faithlife/CodingGuidelines/conventions/dotnet-solution-items-common
```
